Basics of the Web

The web is a collection of HTML documents.
HTML = Hyper Text Markup Language.
HTML is the Main type of document.

HTML

- Text content (what you see)
- Markup (what it looks like)
- References to other documents
- Links to other pages

HTML Markup Tags

Element = <NAME> CONTENTS </NAME>

Bold Tag = <b>CONTENTS</b>

M Tag (Italics/Empahsis) = <em>CONTENTS</em>

HTML Attributes

<Tag Attr="value">CONTENTS</Tag>
Example = <span style="color: #000000">M Tag (Italics/Empahsis) = </span>

Anchor Tag <a> are for links
Example = <a href="www.reddit.com"> Click here for Reddit </a>

Image Tag <img> for images (no end tag as this is a void tag)
Example = <img src="url" alt="text">

 White Space - <br> <p>

In Line Elements - <br> <b> <img> <em>

Block Elements - <p> (creates an invisible box having height and width)

Span / Div

SPAN is inline and DIV is block

<span class = "foo">CONTENTS</span>

<div class = "bar">CONTENTS</div>

HTML Documents

<!DOCTYPE HTML>

 <html>

<head>

<title>CONTENTS</title>

 </head>
